tidy::head

tidy_get_head

(PHP 5, PHP 7, PECL tidy 0.5.2-1.0.0)

tidy::head -- tidy_get_head — Returns a tidyNode object starting from the <head> tag of the tidy parse tree

Description

Object oriented style

tidyNode tidy::head ( void )

Procedural style

tidyNode tidy_get_head ( tidy $object )

Returns a tidyNode object starting from the <head> tag of the tidy parse tree.

Parameters

object

The Tidy object.

Return Values

Returns the tidyNode object.

Examples

Example #1 tidy::head() example

<?php
$html = '
<html>
  <head>
    <title>test</title>
  </head>
  <body>
    <p>paragraph</p>
  </body>
</html>';

$tidy = tidy_parse_string($html);

$head = $tidy->head();
echo $head->value;
?>

The above example will output:

<head>
<title>test</title>
</head>

Notes

Note: This function is only available with Zend Engine 2 (PHP >= 5.0.0).
