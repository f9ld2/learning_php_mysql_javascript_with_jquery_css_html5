APCUIterator::key

(PECL apcu >= 5.0.0)

APCUIterator::key — Get iterator key

Description

public string APCUIterator::key ( void )

Gets the current iterator key.

Parameters

This function has no parameters.

Return Values

Returns the key on success, or FALSE upon failure.
