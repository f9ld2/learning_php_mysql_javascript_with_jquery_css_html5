IteratorIterator::rewind

(PHP 5 >= 5.1.0, PHP 7)

IteratorIterator::rewind — Rewind to the first element

Description

public void IteratorIterator::rewind ( void )

Rewinds to the first element.

Parameters

This function has no parameters.

Return Values

No value is returned.
