MongoCursor::valid

(PECL mongo >=0.9.0)

MongoCursor::valid — Checks if the cursor is reading a valid result.

Description

public bool MongoCursor::valid ( void )

Parameters

This function has no parameters.

Return Values

TRUE if the current result is not null, and FALSE otherwise.
