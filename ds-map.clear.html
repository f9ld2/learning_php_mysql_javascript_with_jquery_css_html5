Ds\Map::clear

(PECL ds >= 1.0.0)

Ds\Map::clear — Removes all values.

Description

public void Ds\Map::clear ( void )

Removes all values from the map.

Parameters

This function has no parameters.

Return Values

No value is returned.

Examples

Example #1 Ds\Map::clear() example

<?php
$map = new \Ds\Map([
    "a" => 1,
    "b" => 2,
    "c" => 3,
]);
print_r($map);

$map->clear();
print_r($map);
?>
