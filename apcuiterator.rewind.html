APCUIterator::rewind

(PECL apcu >= 5.0.0)

APCUIterator::rewind — Rewinds iterator

Description

public void APCUIterator::rewind ( void )

Rewinds back the iterator to the first element.

Parameters

This function has no parameters.

Return Values

No value is returned.
