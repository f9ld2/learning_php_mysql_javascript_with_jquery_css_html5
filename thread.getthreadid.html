Thread::getThreadId

(PECL pthreads >= 2.0.0)

Thread::getThreadId — Identification

Description

public integer Thread::getThreadId ( void )

Will return the identity of the referenced Thread

Parameters

This function has no parameters.

Return Values

A numeric identity

Examples

Example #1 Return the identity of the referenced Thread

<?php
class My extends Thread {
    public function run() {
        printf("%s is Thread #%lu\n", __CLASS__, $this->getThreadId());
    }
}
$my = new My();
$my->start();
?>

The above example will output:

My is Thread #123456778899
