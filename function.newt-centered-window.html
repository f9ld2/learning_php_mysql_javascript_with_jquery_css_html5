newt_centered_window

(PECL newt >= 0.1)

newt_centered_window — Open a centered window of the specified size

Description

int newt_centered_window ( int $width , int $height [, string $title ] )

Open a centered window of the specified size.

Parameters

width

Window width

height

Window height

title

Window title

Return Values

Undefined value.
